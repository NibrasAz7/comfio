# Tutorials

Step-by-step Jupyter notebooks covering the full comfio workflow, from basic compliance contracts to advanced ML/DL integration.

---

## Notebooks

1. **[First Compliance Contract](01_first_contract.ipynb)** — Load sensor data → evaluate 4 domains → Global IEQ → compliance report → contract JSON
2. **[New Domain Modules](02_new_domains.ipynb)** — Pollutant IAQ, sPMV, adaptive comfort, TSV augmentation, personalisation
3. **[Training a Personal Comfort Model](03_training_pcm.ipynb)** — IEQTimeSeriesDataset → PyTorch DataLoader → train PCM → evaluate
4. **[LLM Diagnostics](04_llm_diagnostics.ipynb)** — Tool schemas, function calling, markdown summaries, diagnostic prompts
5. **[Advanced Domain Evaluation](05_advanced_domains.ipynb)** — Daylighting, color quality, reverberation, speech intelligibility, ventilation, psychrometrics
